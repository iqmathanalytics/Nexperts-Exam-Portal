We are building a professional AI-powered Exam Management & Proctored Certification Platform called “NExperts Certification Portal”.

This platform is not a simple LMS. It is a secure enterprise-grade examination system where candidates can:
- Register
- Verify email using OTP
- Login securely
- Purchase exams
- Apply vouchers
- Attend AI-proctored exams
- View results instantly
- Download certificates after passing

There are two major roles:
1. Candidate/User
2. Admin

The platform will later include:
- AI-generated exams
- Voucher management
- Payment management
- Proctoring monitoring
- Result analytics
- Certificates
- Reports
- CSV exports
- Invoice system

For now, ONLY build the FRONTEND UI with clean architecture and mock data.

Tech Requirements:
- React + TypeScript
- Tailwind CSS
- Modern component architecture
- Fully responsive
- Professional SaaS dashboard design
- Dark/light mode
- Clean routing structure
- Reusable UI components

Design Expectations:
- Premium enterprise feel
- Modern certification platform look
- Elegant cards and tables
- Smooth animations
- Excellent UX
- Professional color palette
- Proper spacing and typography
